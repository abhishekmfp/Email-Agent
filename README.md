# Email Agent

## Vision

Build an AI-powered email assistant that understands natural language, drafts professional emails, asks for user approval, and sends emails securely through Gmail.

---

## Problem Statement

Writing repetitive emails is time-consuming. This project aims to automate email composition while ensuring that no email is sent without explicit user confirmation.

---

## Version 1 Goals

- Understand natural language requests.
- Extract recipient, subject, and message.
- Generate a polished email.
- Display a preview.
- Send the email after user approval.

---

## Future Goals

- Contact management
- Calendar integration
- Attachments
- Email scheduling
- Conversation history
- Multi-agent workflow
- Outlook support
- Memory and personalization
